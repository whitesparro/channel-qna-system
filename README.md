# Channel-Based Programming Q&A Tool

A discussion platform where users can create channels, post programming questions, reply in threaded discussions, upload screenshots, and vote on answers.

## Tech Stack
- Next.js
- Node.js API
- PostgreSQL
- Docker

## Setup (Development)

Clone repo:

git clone <repo-url>

Start project:

docker-compose up

## Features
- Channels
- Posts
- Threaded replies
- Voting
- Screenshot uploads
- Search
